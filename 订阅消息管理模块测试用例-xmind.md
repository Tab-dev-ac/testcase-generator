### F01 - 订阅列表展示

#### TC-F01-001 - 列表页展示所有订阅记录
- 优先级: P0
- 前置条件: 安全管理员或应用安全负责人登录系统
- 测试步骤:
  1. 进入"监控告警中心 > 订阅消息管理 > 订阅管理"
  2. 查看页面展示的订阅列表
- 预期结果: 显示所有订阅配置记录，字段包括复选框、订阅名称、订阅应用、订阅方式、订阅人、状态、创建时间、操作
- 状态: 待执行

- 优先级: P1
- 前置条件: 订阅记录超过10条

  1. 进入订阅管理页面
  2. 查看分页组件
  3. 查看列表显示的记录数
- 预期结果: 默认每页显示10条记录，分页组件显示当前页码和总页数
- 状态: 待执行

#### TC-F01-003 - 切换每页显示条数
- 优先级: P1
- 前置条件: 订阅管理页面有数据

### F02 - 新增订阅

#### TC-F02-001 - 新增订阅-填写所有字段保存成功
- 优先级: P0
- 前置条件: 安全管理员登录系统
- 测试步骤:
  1. 进入订阅管理页面
  2. 点击"新增订阅"按钮
  3. 填写订阅名称（如："测试订阅"）
  4. 选择订阅应用（如："应用A"）
  5. 勾选订阅方式（如："钉钉"）

#### TC-EX-001 - 网络请求失败-新增订阅
- 优先级: P1
- 前置条件: 模拟网络断开或服务器错误
- 测试步骤:
  1. 进入订阅管理页面
  2. 点击"新增订阅"
  3. 填写表单
  4. 点击"保存"
- 预期结果: 提示"网络错误，请稍后重试"，弹窗不关闭，可以重新尝试
- 状态: 待执行

- 状态: 待执行

#### TC-EX-003 - 网络请求失败-删除订阅
