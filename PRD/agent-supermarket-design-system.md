## 1. 系统概述

智能体超市系统是承接智能体管理系统数据的下游展示与共享授权平台，核心功能包括：

| 功能模块     | 说明                                   |
|------------|--------------------------------------|
| 智能体目录   | 分类展示、搜索筛选、状态标记                |
| 共享申请     | 申请提交、草稿保存、申请编辑                |
| 审批流程     | 多级审批、驳回/通过操作                   |
| 授权管理     | AK/SK 凭据发放、访问清单、使用记录追溯      |
| 个人中心     | 个人申请记录、我的授权列表                 |

**页面分辨率：** 1920 × 1080px（PC 端），内容区宽度 1200px  
**技术栈：** React 18 + TypeScript + Tailwind CSS v4 + React Router  
**图标库：** Lucide React  

---

## 2. 设计原则

| 原则 | 说明 |
|------|------|
| **一致性** | 所有页面使用统一的颜色令牌、间距单位和组件规范 |
| **清晰性** | 信息层级分明，主次文本颜色区分，辅助色不喧宾夺主 |
| **效率性** | 所有信息模块默认全部铺开展示，Tab 仅作锚定定位 |
| **可控性** | 操作按钮区域固定，状态变化有明确的视觉反馈 |
| **安全感** | 危险操作（释放授权/驳回）均设置二次确认对话框 |

## 4. 字体排版

### 4.1 字体族

```css
font-family: 'PingFang SC', -apple-system, BlinkMacSystemFont,
             'Segoe UI', 'Helvetica Neue', Arial,
             'Noto Sans SC', 'Microsoft YaHei', sans-serif;
```

**规则：** 所有文本均使用 PingFang SC，无论层级。英文数字使用同一字族自动 fallback。

### 4.2 字号规范

| 等级   | 字号    | CSS 变量            | 字重      | 典型用途                  |
|--------|--------|--------------------|---------|-----------------------|
| 超大号   | 32px   | `--font-size-4xl`  | 600     | 数据统计大数字              |
| 大标题   | 24px   | `--font-size-3xl`  | 600     | 页面主标题（h1）            |
| 次标题   | 20px   | `--font-size-2xl`  | 600     | 卡片主标题（h2）            |
| 小标题   | 18px   | `--font-size-xl`   | 600     | 模块标题（h3）             |
| 正文大   | 16px   | `--font-size-lg`   | 400/600 | 卡片名称、Tab、正文段落      |
| 正文基础  | 14px   | `--font-size-base` | 400     | 表格内容、表单字段、描述文字  |
| 辅助说明  | 13px   | `--font-size-sm`   | 400     | 表单 Label、次要信息       |
| 极小号   | 12px   | `--font-size-xs`   | 400/500 | 标签文字、时间戳、徽章       |

### 4.3 字重

| 名称    | 数值  | CSS 变量                  | 用途                |
|--------|------|--------------------------|-------------------|
| Regular | 400  | `--font-weight-regular`  | 正文、辅助说明         |
| Medium  | 500  | `--font-weight-medium`   | 按钮、标签激活态       |
| SemiBold| 600  | `--font-weight-semibold` | 标题、模块名称、数字    |
| Bold    | 700  | `--font-weight-bold`     | 强调数字（可选）       |

### 4.4 行高

| 名称     | 值     | CSS 变量               | 用途          |
|---------|-------|----------------------|-------------|
| tight   | 1.2   | `--line-height-tight`  | 大标题、数字展示  |
| base    | 1.5   | `--line-height-base`   | 正文内容       |
| relaxed | 1.75  | `--line-height-relaxed`| 多行描述、富文本  |

## 5. 间距规范

系统采用 **4px 基准网格**，所有间距为 4 的倍数：

| 令牌        | 值     | CSS 变量      | 典型用途                  |
|-----------|-------|-------------|------------------------|
| `space-1` | 4px   | `--space-1` | 最小间距、图标与文字间距         |
| `space-2` | 8px   | `--space-2` | 标签内边距、紧凑元素间距         |
| `space-3` | 12px  | `--space-3` | 按钮左右内边距（sm）、表格行高    |
| `space-4` | 16px  | `--space-4` | 表格单元格、表单项间距          |
| `space-5` | 20px  | `--space-5` | 卡片标题与内容间距             |
| `space-6` | 24px  | `--space-6` | 卡片内边距、主要元素间距         |
| `space-7` | 28px  | `--space-7` | 区块间距（辅助）              |
| `space-8` | 32px  | `--space-8` | 页面模块间距、大按钮内边距        |
| `space-10`| 40px  | `--space-10`| 页面顶部/底部留白            |
| `space-12`| 48px  | `--space-12`| 页面大区块间距               |

## 7. 布局规范

### 7.1 页面尺寸

- **设计稿尺寸：** 1920 × 1080px
- **内容区最大宽度：** `1200px`（水平居中）
- **内容区左右内边距：** `24px`
- **顶部导航栏高度：** `56px`（固定吸顶）
- **页面顶部留白：** `32px`（breadcrumb 上方）
- **卡片间距：** `24px`

### 7.2 网格

- **智能体卡片网格：** 3 列等宽，间距 24px
- **信息展示网格：** 2 列，列间距 64px，行间距 16px
- **筛选条件网格：** 3 列，间距 16px
- **统计卡片网格：** 4 列，间距 16px

### 7.3 Z-Index 层级

```
页面内容     z-index: 0
Sticky 导航  z-index: 1020
Fixed 元素  z-index: 1030
下拉菜单    z-index: 1000
Modal 遮罩  z-index: 1050
Popover     z-index: 1060
Tooltip     z-index: 1070
```

## 10. 交互流程

```
智能体目录
    │
    ├── 点击卡片（未申请）
    │       └── 智能体详情页
    │               └── 点击「申请共享」
    │                       └── 申请编辑页（/application/form/:id）
    │                               ├── [保存草稿] → 保留草稿，留在当前页
    │                               ├── [提交申请] → 成功Toast → 返回目录
    │                               └── [返回] → 返回详情页
    │
    ├── 点击卡片（申请中）
    │       └── 智能体详情页（申请中状态）
    │               └── 含申请信息 + 审批流程模块
    │                   └── 申请详情审批页（只读）
    │                           ├── [驳回] → 确认弹窗 → 填写理由 → 驳回
    │                           ├── [通过] → 确认弹窗 → 通过
    │                           └── [返回] → 返回详情页
    │
    └── 点击卡片（已授权）
            └── 智能体详情页（已授权状态）
                    └── 含申请信息 + 授权记录 + 使用记录 + 审批流程
                        └── 点击「释放授权」
                                └── 确认对话框
                                        ├── 备注说明（选填）
                                        ├── [取消] → 关闭弹窗
                                        └── [确认释放] → 成功Toast → 返回目录
```
